## Features

- Browser-based retro game emulator
- Support for NES, SNES, N64, PlayStation, and 30+ classic systems
- Drag & drop ROM support
- PWA support (install as app)
- 7 switchable theme styles

## Themes

| Theme | Description |
|-------|-------------|
| Steamboat Willy | Pure black & white monochrome |
| Sin City | Black & white with blood red accents |
| Dark Night | Pure grayscale, minimal |
| Grindhouse | Dark with red/yellow punches |
| Dusk Till Dawn | Deep crimson reds |
| Kill Bill | Yellow and black |
| Cyberpunk | Neon magenta/cyan/green |

## Usage

1. Visit the live site
2. Select a theme from the top bar
3. Drag & drop a ROM file or browse your library
4. Press Start and play!

## Supported Systems

NES, SNES, Game Boy, Game Boy Color, Game Boy Advance, Nintendo 64, Nintendo DS, Sega Genesis, Sega CD, Sega Saturn, Sega Dreamcast, PlayStation, PSP, Atari 2600, Atari 7800, Neo Geo, TurboGrafx-16, and more.
